# AERIS Backend (V2 Microservices Architecture)

> **Note for Judges/Reviewers:**
> The `backend/` directory contains our **V2 Asynchronous Microservices Architecture** (FastAPI, Celery, PostGIS).
> 
> For the **Build With AI Hackathon MVP**, we are running the **Serverless V1 Architecture** (Next.js Edge API routes hitting Gemini directly) to maximize deployment speed on Vercel. 
> 
> You **do not** need to run `docker-compose up` or configure this Python backend to test the application or use the frontend Command Center. The entire MVP is self-contained in the `frontend/` Next.js application!
